# Landing Page

Modern marketing landing page with animated hero, feature grid, pricing tables, testimonials, newsletter signup, and responsive nav/footer. Built with Next.js 14, Tailwind CSS, and lucide-react. Uses {{BRAND_NAME}}, {{TAGLINE}}, {{DESCRIPTION}}, {{CTA_TEXT}}, {{PRIMARY_COLOR}} placeholders.

**Stack:** Nextjs | **Complexity:** simple
**Tags:** `landing`, `marketing`, `hero`, `pricing`, `testimonials`, `footer`, `newsletter`, `responsive`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
